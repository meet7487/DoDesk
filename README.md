# 🚀 DoDesk – Project Management Platform

DoDesk is a full-stack project management web application designed to help teams manage tasks, projects, and workflows efficiently. It provides an intuitive interface, real-time updates, and structured task organization to boost productivity.

---

## 📌 Features

* ✅ Create and manage projects
* ✅ Task creation with status tracking (Start → Complete)
* ✅ Drag-and-drop task management (Kanban style)
* ✅ Deadline-based task categorization
* ✅ Team collaboration and member assignment
* ✅ Real-time data updates using Firebase
* ✅ Responsive and modern UI

---

## 🛠 Tech Stack

### Frontend

* React.js
* Material UI
* CSS

### Backend

* FastAPI
* MongoDB

### Other Tools

* Firebase Authentication
* Firestore (Realtime Database)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/meet7487/DoDesk.git
cd DoDesk
```

---

### 2️⃣ Setup Frontend

```bash
cd Dodesk_Frontend
npm install
npm run dev
```

---

### 3️⃣ Setup Backend

```bash
cd Dodesk_Backend
pip install -r requirements.txt
uvicorn main:app --reload
```
